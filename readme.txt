チームで開発した格闘ゲームで、Unityを使用しています。
4か月で開発が終わり、現在担当した箇所のコードをリファクタリングしています。

担当したスクリプト

Scripts>GameScene>OptionのOptionButton、CommandButton

GameScene>PlayerのAnimationState、ColAnimationEvent、PlayerAction、PlayerAnimationEvent、
PlayerGuard、PlayerGuardPose、PlayerInputSystem

SelectModeSceneのChooseMode、SelectModeScene

SelectStageSceneのSelectStageScene

SoloSceneのCarDamage、FinishOption、PauseMenu、Score、SoloModeCountDown、UIAnimation

TitleSceneファイルの中身全部

TutorialSceneファイルの中身全部

注力した箇所
InputSystemを使って、キーボードとコントローラーのどっちで操作しても違和感がないようにした

格ゲー特有のコンボを実装させるためのキャンセルフレームの実装
攻撃や移動のアニメーションを再生する前に、一旦全部のアニメーションをデフォルトに戻すことによって実装した
アニメーション開始→アニメーションイベントでキャンセルできるフレームになる→ボタンを押す→
実行されていたアニメーションがデフォルト状態に強制的に戻る→押したボタンのアニメーションが実行される
